A simple and interactive web-based Student Marks Management System built using HTML, CSS, and JavaScript. This project helps faculty manage student marks efficiently with automatic calculations, charts, and PDF reports.
🚀 Features
🔐 Login Authentication (Admin access)
📋 Student List Display
✍️ Marks Entry (A1, A2, MCQ1, MCQ2)
📊 Automatic IAT 60% Calculation
➕ Boost Marks Entry
🧮 Total Marks & Grade Calculation
🥇 Topper Highlight
🌙 Dark Mode
📈 Bar Chart & Pie Chart Visualization
📄 PDF Download (All Subjects in One File)
📊 Pass / Fail / Average Calculation
🛠️ Technologies Used
HTML
CSS
JavaScript
Chart.js
jsPDF
jsPDF AutoTable
